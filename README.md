# Mudman Blues Website

This is a full-stack web application designed to host my musical discography and lyrics. It serves as a centralized hub for listeners to explore the project’s history and access songs via integrated YouTube links.

### Live Link
[https://mudman-blues.onrender.com](https://mudman-blues.onrender.com)
*Note: Hosted on Render's free tier; the initial load may take a moment to wake up the server.*

### Tech Stack
* **Backend:** Java (Spring Boot), JDBC
* **Database:** PostgreSQL
* **Frontend:** HTML, CSS, JavaScript
* **Infrastructure:** Docker, Render

### Features
* Dynamic search and filtering of song lyrics and metadata.
* Integrated YouTube links for all tracks.
* Relational database management using Java JDBC and SQL.
* Secure credential management via Environment Variables.
